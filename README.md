# Coffee Sales Excel Dashboard

An Excel-based sales analysis dashboard built using real-world style coffee shop data.

<img width="940" height="389" alt="image" src="https://github.com/user-attachments/assets/2382226e-2c08-4760-b94b-98e9cca26781" />

## What's in it

- **Dashboard** — Visual summary of key metrics
- **Total Sales** — Monthly sales broken down by coffee type
- **Sales vs Country** — Which country generates the most revenue
- **Sales vs Customer** — Top customers by purchase value
- **Raw Data** — Orders, Customers, and Products sheets

## Tools Used

- Microsoft Excel
- Pivot Tables
- Charts & Dashboard design

## Data Overview

| Sheet | Description |
|-------|-------------|
| ORDERS-DATA | 1000 orders with customer, product, quantity, sales |
| CUSTOMER-DATA | Customer info — name, country, loyalty card |
| PRODUCT-DATA | Coffee types, roast, size, price, profit |

## Key Insights

- United States is the highest revenue country
- Arabica and Liberica are top-selling coffee types
- Loyalty card customers show repeat purchase patterns

## How to use

Download the `.xlsx` file and open in Excel (2016 or later recommended for full pivot/slicer support).
